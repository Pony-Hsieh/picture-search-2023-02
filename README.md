## 開發指令相關
- Project Setup
    ```sh
    npm install
    ```
- Compile and Hot-Reload for Development
    ```sh
    npm run dev
    ```
- Type-Check, Compile and Minify for Production
    ```sh
    npm run build
    ```
- 使用 eslint 檢查語法，並自動修復
    ```sh
    npm run lint-fix
    ```
